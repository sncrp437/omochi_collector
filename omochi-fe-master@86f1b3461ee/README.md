# Omochi

An application for ordering food through QR codes at the table. Built with React, TypeScript, Redux, and Tailwind CSS.

## Features

- Scan QR code to access the menu
- View list of food items by category
- Add food items to cart
- Online payment
- View order history
- ...

## Technologies Used

- React + TypeScript
- Redux Toolkit
- React Router
- Tailwind CSS
- Vite
- Ant Design

## Requirements

- Node.js 20 or higher
- npm or yarn

## Installation

1. Install dependencies:
```bash
npm install
```

2. Set up openapitools
In the openapitools.json file, change the path in the inputSpec field to your backend API path
```bash
npm run generate-api
```

3. Run the project in development environment:
```bash
npm run dev
```

4. Build the project for production:
```bash
npm run build
```
